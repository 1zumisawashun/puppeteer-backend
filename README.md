## 採用している技術

```
言語：TypeScript
サーバー：Express
トークン：JsonWebToken
スクレイピング：Puppeteer
DB：Firestore
ストレージ：Cloud Storage
フロント：EJS
```

## 直面している課題

- model の作成＞ firebase の converter を導入する
- CSS から Sass へ変更する
- Vercel へのデプロイ＞ API サーバーとしても活用する
- まだ any になっている箇所の修正
