# puppeteer-backend

サーバー：express  
セキュリティ：jwt・cookie-parser・bcrypt・express-validator  
スクレイピング：puppeteer-core  
データベース：firestore  
フロント：ejs

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm start
```

main app.js now

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
